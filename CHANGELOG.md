# Changelog

All notable changes to this project will be documented in this file.

## 0.0.0

### Added

- Initial release of Simple AgentCore Construct
- Support for deploying AWS Bedrock AgentCore Runtime with Docker containers
- Automatic ECR repository creation and image deployment
- IAM role creation with required Bedrock AgentCore permissions
- Service-linked role management for network and runtime identity
- Multi-language support: TypeScript, JavaScript, Python (via jsii)
